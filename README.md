# Windhawk Mods

A collection of [Windhawk](https://windhawk.net/) mods for Windows 11.

## Mods

### [Taskbar Media Beacon](taskbar-media-beacon.wh.cpp)

A native-style now-playing widget for the Windows 11 taskbar.

![Taskbar Media Beacon](https://raw.githubusercontent.com/jihedkdiss/Windhawk-Mods/refs/heads/main/taskbar-media-beacon.png)
